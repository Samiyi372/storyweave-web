# Storyweave - 城市遗产沉浸式步行导览

Storyweave 产品介绍官网，Storyweave 移动端 App 的 Web 端配套展示页。

Storyweave 是一款基于 GPS 触发的移动端音频导览平台，通过分支叙事结构将城市中被忽视的历史人物故事带入真实街道，让用户在行走中沉浸式体验城市遗产。

## 技术栈

- 纯 HTML / CSS / JavaScript（无框架、无构建工具）
- Google Fonts：Bricolage Grotesque、Inter
- 单文件结构（`index.html`）

## 本地启动

无需安装任何依赖，直接用浏览器打开即可：

```bash
# 克隆仓库
git clone https://github.com/zhitian-h/storyweave-web.git

# 进入目录，用浏览器打开
open index.html
# 或 Windows 下：start index.html
```
